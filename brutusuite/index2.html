⬇ Get started

Install in 2 minutes

BrutuSuite is open‑source and runs directly from source. No store required — just clone and load.

1

Clone the repo

git clone https://github.com/hangga/brutusuite.git

2

Open Chrome Extensions

Navigate to chrome://extensions/ and enable Developer mode.

3

Load unpacked

Click Load unpacked and select the folder containing manifest.json.

BrutuSuite v1.0

DevTools extension · open source

$ git clone https://github.com/hangga/brutusuite.git $ cd brutusuite $ # load unpacked in chrome://extensions/

🧱 Under the hood

Simple, modular architecture

Three components work together to give you seamless network visibility inside Chrome DevTools.

⚙️

Background Service

Uses the Chrome debugger API to capture network events and store logs.

background.js
🖥️

DevTools Panel

Displays requests, details, and the edit/replay interface. Communicates with the background script.

panel.html · panel.js
🔌

DevTools Entry

Registers the BrutuSuite panel inside Chrome DevTools.

devtools.js
